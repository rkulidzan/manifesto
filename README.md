# Manifesto Template

A free, fillable manifesto template created by [Rebekah Kulidzan](https://rkulidzan.com).

Built as a companion to the blog post [The Manifesto](https://rkulidzan.com/blog/) on why writing one matters, especially during career transitions.

## What it is

A single-page interactive template for working out what you actually want from work. Five sections:

1. What do you need from work to function well?
2. What are your non-negotiables?
3. What does belonging look like for you?
4. What is your thread?
5. What do you commit to yourself?

Fill it in your browser and save as PDF, or print directly.
